# Summer Writing '26 — Design System

A neo-brutalist visual system for a 30-day daily writing challenge. The site lives at **writing.amiralsad.blog/summer-2026** and consists of two page types:

1. **Landing page** — title, description, topics component, and a vertical feed of published pieces (one new piece goes live each day).
2. **Post page** — date, title, description, reading-time, body content in a centered reading column, and prev/next navigation.

The site renders markdown files filtered by date.

---

## The vibe in one paragraph

This is a **personal publication, dressed loud.** It looks like a hot-pink newsstand zine printed on cream paper, designed by someone who loves _Grand Theft Auto: Vice City_, mid-century Penguin paperbacks, and the chunky terminal type used by software companies like **Greptile**. Display headlines are massive condensed grotesks set tight. Body copy is a generous reading-serif. Every interactive element sits on a **hard offset shadow** — no blur, just a flat ink-navy block behind it. Color is used like a highlighter, not a wash: turquoise, coral, hot-pink, and sun-yellow against the cream paper.

## Inspiration

The brand converges three references:

- **Tavus** (https://www.tavus.io) — the rectangular window-chrome cards with tab strips, the hard offset shadows on every CTA, the serif headline + sans-mono eyebrow combo, the pixelated-dithered hero imagery. See `references/tavus-hero.png`, `references/tavus-features.png`.
- **Greptile** (https://www.greptile.com) — the heavy condensed display sans set tight, the hexagonal/pointed pill buttons, the grid-paper background, the navy + neon green accent. See `references/greptile-hero.png`, `references/greptile-how.png`.
- **Miami / Vice City** illustrations — the color palette. Turquoise sea, coral sunset, peach clouds, dusty rose, deep navy ink. See `assets/vice-city-*.png`.

We landed on: **Greptile's typographic confidence** + **Tavus' interactive-window chrome** + **Vice City's palette** + **cream paper** as the surface.

## CONTENT FUNDAMENTALS

Copy is written like the author is talking to a friend who reads a lot. No corporate softness. No exclamation points. No emoji.

- **Voice.** First-person singular for posts ("I spent the week trying to..."). Second-person for the landing page meta-copy ("Read along, or write your own."). Confident but not braggy. Comfortable with a fragment.
- **Casing.** Sentence case for everything except metadata, which is `UPPERCASE MONO` (`DAY 14 · 06.14.26 · 7 MIN`). No Title Case headlines — they look like marketing.
- **Length.** Titles are short, ideally under 8 words. Descriptions are one sentence, one comma at most. Body posts run 800–2000 words.
- **Punctuation.** Em-dashes are encouraged. Oxford comma, always. Italics for emphasis, not bold.
- **Numbers.** Spell out one through nine; numerals for 10+. Dates as `06.14.26` in mono metadata, or "June 14" in prose.
- **No emoji. No ✨. No 🚀. No corporate sparkle.** If you want emphasis, use italics or a colored highlight.
- **No filler.** The site has two page types, period. Don't invent an "About" page unless asked.

### Copy examples (use these as the canonical voice)

**Landing description (canon):**
> Thirty pieces in thirty days. Some essays, some fiction, some half-formed. New writing every morning through June.

**Topic chip labels:**
> essays · fiction · notes · marginalia · ephemera

**Post title examples:**
> "The argument I keep losing"
> "Notes from a porch in Naples"
> "On giving up on a novel"

**Post description (one sentence):**
> "Why I can't stop rereading the same forty pages of _Stoner_."

**Metadata format:**
> `DAY 14 · 06.14.26 · 7 MIN READ`

**Prev/next link copy:**
> `← PREVIOUS · The argument I keep losing`
> `NEXT · A porch in Naples →`

**Empty state for unpublished days:**
> "Not yet."

## VISUAL FOUNDATIONS

### Colors

The palette is a Miami sunset projected onto cream paper. Pick **one** color per surface — never mix two brights at full strength in the same card.

- **Cream paper** `#F5EFE3` — the default background, everywhere
- **Deep navy ink** `#1B1F3A` — text, borders, shadows. **Never use pure black.**
- **Hot pink** `#FF3E80` — the primary CTA color, the signature highlight
- **Turquoise** `#5BD6C3` — secondary accent, link hover background, surf
- **Coral** `#FF8B6A` — tertiary accent for chips/eyebrow markers
- **Peach** `#F4B0B8` — soft fills, alert/banner bands
- **Mint** `#B7F0C8` — success / "live" indicator
- **Sun yellow** `#FFC857` — sparingly, for the day-counter chip

White (`#FFFFFF`) is reserved for **window-chrome panels** (Tavus-style card frames). Everything else sits on cream.

### Type

- **Big Shoulders Display** (700/800/900) — display headlines, big poster type, button labels. Condensed, tight, heavy, all-caps optional.
- **DM Serif Display** (Regular + Italic) — editorial subheads, quotes, the "What if…" rhetorical question style.
- **Source Serif 4** (Regular + Italic, optical sizing) — long-form body text in the reading column.
- **JetBrains Mono** (500/700) — dates, eyebrows, day counters, codes, prev/next metadata.

Body sets at **18–20px** with **1.55–1.7 line height** for the reading column. Headlines run from 48px up to ~200px on the landing poster.

### Backgrounds

- The default page background is **cream paper** (`#F5EFE3`). Always.
- A **dot-grid overlay** (1px ink dots, 24px spacing, ~12% opacity) sits behind the landing hero — call it "graph paper."
- Imagery (when used) gets a **halftone/dither treatment** — black-and-white halftone over a flat color block. We don't use gradients as backgrounds. The only "gradient" we have is the **sunset peach → turquoise** vertical wash, used once on the landing footer.
- Full-bleed illustrations (the Vice City reference art) are reserved for the landing hero illustration slot — never inside post body.

### Animation

- **Restrained.** Page elements don't fly in. Hover states are instant.
- **Cards on hover** translate `-2px, -2px` and the shadow grows from `5px 5px` to `8px 8px` over `200ms ease-out`. Pressed = translate `+2px, +2px`, shadow shrinks to `2px 2px`.
- **Day counter** has a single tasteful 1-second "blink" on page load only.
- No bounces, no spring overshoot, no scroll-jacking, no parallax.

### Hover & press states

- **Buttons.** Hover: shadow grows, button shifts up-left by 2px. Press: button shifts to where shadow was, shadow disappears (`0 0` offset). This is the brutalist standard.
- **Links.** Hover: background fills with hot pink, text becomes paper-cream, underline removed.
- **Cards / post-feed rows.** Hover: the entire row shifts 8px to the right, the **day number** flips to hot pink and bumps `-2px,-2px`, the **title** color shifts to hot pink, and a small hot-pink arrow slides in next to the reading-time chip. No underline, no background wash.

### Borders

- Default border is **3px solid navy ink** (`var(--bw-2)`).
- Heavy poster borders are **6px**.
- Hairlines (table dividers, post-feed separators) are **1px solid navy at 18% opacity** (`--rule-soft`).
- Sharp corners are the default. `border-radius: 0`. The only rounded thing is the **pill button** (capsule), and the **window-chrome card corners** (2px / 6px).

### Shadow system

- **No blur. Ever.** Every shadow is a hard offset block in navy ink.
- Levels: `3px 3px`, `5px 5px`, `8px 8px`, `12px 12px`. Pick by importance.
- For "loud" CTAs we sometimes color the shadow hot-pink or turquoise instead of ink — see `--sh-pink`, `--sh-turq`.

### Layout rules

- **Max width 1200px** for the landing container; **680px** for the reading column.
- Page header is **not fixed**. It scrolls with content. (We are a publication, not an app.)
- The day counter chip in the post header is **sticky-top** with a 20px offset, sitting just below where the page header was.
- Grid alignment is loose — we let things hang into the margin on purpose, especially the poster headline on the landing page.

### Transparency & blur

- We don't use frosted glass / backdrop-filter blur. Period.
- Transparency is only used for the dot-grid overlay and rule-soft hairlines.

### Imagery treatment

- **Black-and-white halftone** over a flat color rectangle. Cool/warm bias depending on the post mood — but we lean warm overall.
- Photos appear in `<image-slot>` blocks the user fills in; we never auto-generate or AI-illustrate.

### Corner radii recap

- Sharp `0` — cards, panels, post-feed rows, image frames.
- `2px` — window-chrome inner panes, code blocks.
- `6px` — Tavus-style outer window frame.
- `999px` (pill) — buttons, chips, day-counter.

### Cards

The two card types:

1. **Window-chrome card** (Tavus reference). White background, 3px ink border, 6px outer radius, a colored title-bar strip at the top with an eyebrow label. Hard offset shadow.
2. **Post-feed row.** Not a card at all — it's a row in a list with a 1px ink hairline above. Each row has a day-number, title, one-sentence description, reading time. Hover: title gains a 2px underline; day-number chip flips color.

## ICONOGRAPHY

Iconography is **deliberately minimal.** This is a writing site, not a SaaS product.

- We use **Lucide** (https://lucide.dev) via CDN for the handful of icons we need. Stroke width: `2.25px`. Color: `currentColor` (always ink, occasionally turning to paper-cream on a hot-pink fill).
- The icons we actually use: `arrow-left`, `arrow-right`, `book-open`, `pen-tool`, `calendar`, `clock`, `rss`, `chevron-down`, `external-link`. That's the entire approved set.
- **No emoji.** Anywhere.
- **No filled SVG illustrations** other than the brand logo mark and the Vice City reference art slot.
- Unicode geometric shapes (`■ ▶ ▲ ●`) are used as **inline bullet markers** in the eyebrow strip, matching the Tavus reference. Always set in the display font, never in the body serif.

Lucide is loaded via CDN — see `ui_kits/website/index.html`. If you need a glyph that isn't in Lucide, **flag it and ask** — don't draw a custom SVG.

### Brand mark

The mark is a **typographic monogram** — `S/W` in Big Shoulders Display Black, with a thick navy slash separating them. See `assets/logo-sw.svg` (mark) and `assets/logo-sw-wordmark.svg` (mark + "SUMMER WRITING '26" lockup).
